How your data is protected

Everything in this tool happens inside your own browser. Your data is never sent anywhere.

  • Read locally. When you load CSV files, the browser reads them on your device with its built-in file reader. They stay in memory only while you work.
  • Never uploaded. The app makes no network request that carries your data. A strict Content-Security-Policy tells your browser to block any connection to another server.
  • Not stored. Nothing is written to disk, cookies, or browser storage — close the tab and the data is gone. Files you choose to Export are saved to your own computer.
  • No third parties. No analytics, trackers, ads, or external code — just a few small files served as static content.
  • Works offline. Once the page has loaded you can disconnect from the internet and it still works — the simplest proof that nothing needs to be sent.

Like any website, the host sees your browser request the page and your IP address, but never your file contents. Browser extensions you install can read any page you open, so use ones you trust.

Want to verify it yourself? Open your browser's Network tab, load a file, and run a comparison — you'll see no request carrying your data.
